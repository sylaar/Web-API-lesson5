# Анализ зарплат разработчиков (HeadHunter + SuperJob)

Скрипт для анализа рынка IT-вакансий. Собирает данные с HeadHunter и SuperJob, вычисляет среднюю зарплату по языкам программирования.

## Зависимости

`requests` – для HTTP-запросов

`environs` – для загрузки переменных из .env

`tabulate` – для красивого вывода таблицы

## Запуск проекта

1. **Клонировать репозиторий**
```bash
git clone https://github.com/sylaar/Web-API-lesson5.git
cd Web-API-lesson5
```
2. **Установить зависимости**

```bash
pip install -r requirements.txt
```
3. **Настроить API-ключи**
Создать файл .env в корне проекта:

```.env
SUPERJOB_API_KEY=<ВАШ_КЛЮЧ_SUPEJOB.ru>
```
Запустить скрипт

```bash
python main.py
```

## Структура проекта
|Файл|Описание|
| ------------- |:-------------:|
|`main.py` |Основной скрипт, запускает сбор данных и вывод статистики     |
|`hh_api.py`|Функции для работы с API HeadHunter|
|`superjob_api.py`	|Функции для работы с API SuperJob (требует ключ)|
|`predict_salary.py`|Логика расчёта зарплаты из текста вакансии|
|`calculate_statistics.py`|Подсчёт средней зарплаты|
|`print_stats_to_terminal.py`|Вывод результатов в виде таблицы|
|`settings.py`|Конфигурация|

## Что делает программа
Запрашивает вакансии разработчиков на HeadHunter и SuperJob
(Языки программирования: Python, Java, C++, JavaScript, Ruby, C#, Go, PHP, R, Fortran)

Для каждой вакансии вычисляет предполагаемую зарплату (только для вакансий, с зп в рублях)

Собирает статистику по каждому языку:

- название вакансии
- количество найденных вакансий
- количество обработанных вакансий
- средняя зарплата

## Примечания по API
HeadHunter
Публичный API, ограничение – 1 запрос в секунду. Скрипт соблюдает паузы.

SuperJob
Требуется ключ, который можно получить на api.superjob.ru.
